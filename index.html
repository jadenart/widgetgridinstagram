💡 Você precisará de um Notion Integration Token e do Database ID do seu banco de dados de conteúdo. Veja as instruções abaixo.
Como criar o token:
1. Acesse notion.so/my-integrations
2. Clique em "New integration" → dê um nome → salve
3. Copie o Internal Integration Token
4. No seu database, clique em ••• → "Add connections" → selecione sua integration

Como pegar o Database ID:
Na URL do seu database: notion.so/[workspace]/[DATABASE-ID]?v=...
